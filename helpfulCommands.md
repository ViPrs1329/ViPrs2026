### push code to the robot:
```powershell
robotpy deploy
```

### push code without tests:
```powershell
robotpy deploy --skip-tests
```

### simulate robot:
```powershell
robotpy sim
```

### install all helpful robotpy packages:
```powershell
pip install robotpy[all]
```

### install robotpy package:
```powershell
pip install robotpy[package name]
```
